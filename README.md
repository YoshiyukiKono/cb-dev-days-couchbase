### Couchbase Server環境構築

本ワークショップでは、ご自分のローカル環境で、Dockerを用いて、Couchbase Serverを導入・環境設定すること方法と、
講師が準備した、AWS上の環境設定済みのCouchbase Serverへアクセスする方法の二通りから選択することが可能です。
前者を選択される場合は、[こちら](./docs)の手順に従って、実行してください。
